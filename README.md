# Ejercicios de Listas y Tuplas en C#

## Descripción
Este proyecto contiene la resolución de cinco ejercicios de Listas y Tuplas tomados del sitio Aprende con Alf, los cuales han sido implementados en el lenguaje C# utilizando Programación Orientada a Objetos (POO).

## Ejercicios desarrollados
1. Almacenamiento y visualización de asignaturas.
2. Visualización de asignaturas con el mensaje “Yo estudio”.
3. Registro de notas por asignatura.
4. Ordenamiento de números de lotería.
5. Visualización de números del 1 al 10 en orden inverso.

## Tecnologías utilizadas
- Lenguaje: C#
- Entorno: .NET
- Editor: Visual Studio Code
- Control de versiones: GitHub
